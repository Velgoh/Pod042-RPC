# Pod 042: YouTube Music Discord RPC

A sleek, automated Discord Rich Presence (RPC) tool themed around Pod 042 from *NieR:Automata*. This tool automatically reads what you are listening to on YouTube Music and broadcasts it beautifully to your Discord profile.

## Features
* **Zero Dependencies:** You do not need to install Node.js! The runtime is bundled directly inside so it works right out of the box.
* **Auto-Start Integration:** Automatically launches silently in the background when you turn on your PC.
* **Start Menu Shortcuts:** Search your Windows Start Menu to easily start or stop the Pod at any time.
* **NieR Theming:** Features custom Pod 042 icons and styling for your Discord profile.

## How to Install
1. Download this entire repository as a folder to a permanent location on your PC (like your Documents folder).
2. Double click `Install.bat`.
3. The script will automatically configure your Windows Startup folder, build the Start Menu shortcuts, and launch the Pod!

## How to Stop
If you want to close the RPC, simply press your Windows key, type **Stop Pod 042**, and press Enter.
